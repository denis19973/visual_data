#Visual Data#

##REQUIREMENTS:##

* Flask 0.12
* MongoDB 3.2.11
* python3.5
* d3.js 3.0
* topojson


##FUNCTIONALITY:##

Simple web-application for visualisation data from database.
Back-end side is wrote on Flask and uses MongoDB.
On the client side AJAX request gets data from server 
and shows using d3.js.
